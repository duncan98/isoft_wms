==============================安装说明==============================

1、虚拟主机根目录新建一个文件夹例如:"sm",将source下的目录及文件完整上传到sm下
2、如果部署环境是linux,请确认App/Runtime目录属性为777
3、假设www.yourdomain.com是您的域名，sm是项目目录，那么请打开http://www.yourdomain.com/sm/install.php进行安装
4、请准备好数据库的相关信息，根据安装向导进行操作，
5、系统默认管理员帐号为admin/admin,普通用户帐号user/user

目录结构：
source           源码(上传此目录下内容)
  └─App         项目目录
	└─Backup  数据库备份目录
	└─Conf    数据库配置目录
	└─Lib     项目类库目录
	└─Runtime 缓存目录
	└─Tpl     模版目录
  └─Public      引用文件目录
  └─Thinkphp    Thinkphp目录
